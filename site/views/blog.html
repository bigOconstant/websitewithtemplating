Run Emacs in WSL

If you're like me you like emacs, but for what ever reason you need to run windows 10.This is a problem for a few reasons.

1.Power Shell is still Garb. It's ugly and doesn't come with grep.

2.Having a terminal in Emacs is my favorite part of emacs.

Getting a shell to work in emacs for windows is a pain

3. Emacs in a wsl terminal still suffers from a few quirks.

So whats the solution?

Install xming, set the display value in wsl and load emacs through the x server

1. Install xming and launch it in windows

Xming is a x-server for windows. You can download it here.

2. In wsl install x11 utils

sudo apt-get install x11-apps

3. In wsl set the display to 0 

export DISPLAY=:0

4. Install emacs launch it, and enjoy a full graphical version of linux emacs on windows
